# qiankun 微前端实践指南

## 快速上手
> node 版本 >= 14.0.0 <= 16.0.0
```
yarn demo:install
yarn demo:start
```
本地访问 [`http://localhost:8080/admin/index.html`](http://localhost:8080/admin/index.html)
![example.gif](./example.gif)

## 特性
- [x] 基于 qiankun 的微前端架构
- [x] 支持多种框架
- [x] 支持多种路由方案
- [x] 支持多页签切换缓存
